# handwritten-digit-recognition-ML

A machine learning project to recognize handwritten digits from 1 to 9 using Python and popular ML libraries.

This project demonstrates data preprocessing, model training,evaluation, and prediction with high accuracy.

## Technologies & Skills
- Python, NumPy, Pandas,matplotlib
- Scikit-learn,
- Data preprocessing and feature engineering
- Model training, evaluation, and prediction
- Jupyter Notebook for experimentation
